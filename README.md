# c-world-encode

h5 helper scripts for encode/dekkerlab hi-c data
convert hdf5 (h5) files into tsv (tab seperated value) text (matrix) files.

```
c-world-encode/
	hdf2tab.py - Extract c-data from HDF5 file into TXT (matrix.gz)
```
